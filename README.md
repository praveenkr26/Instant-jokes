# 😂 AI Joke Generator (Gemini API + Vanilla JS)

A fun and interactive web application that generates jokes based on any topic entered by the user.  
Built using **HTML, CSS, and JavaScript only**, powered by the **Google Gemini API**.

---

## 🚀 Project Overview

This project allows users to type any topic (like “school”, “coding”, “cricket”, “AI”, etc.) and instantly get a **funny AI-generated joke**.

It is a **frontend-only project** that demonstrates API integration without any frameworks.

---

## ⚙️ Tech Stack

- 🌐 HTML
- 🎨 CSS
- ⚡ JavaScript (Vanilla JS)
- 🤖 Google Gemini API

---

## ✨ Features

- 😂 Generate jokes on any topic
- ⚡ Instant AI response
- 🎯 Simple and clean UI
- 📱 Fully responsive design
- 🔥 No frameworks used (pure JS project)
- 🧠 AI-powered humor generation
